Kulala is a powerful API toolset that provides a comprehensive suite of features
to help you develop, test, and manage your APIs efficiently.

This is a list

## Editor integrations

### Neovim

[Kulala.nvim](/apps/kulala.nvim) is the first Kulala editor integration,
currently our flagship app for the ecosystem.

## CLI tools

### Kulala CLI

[Kulala CLI](/apps/kulala-cli) makes it easy to run .http files from the command line,
with support for various features like environment variables, output formatting, and more.

### Kulala Fmt

[Kulala Fmt](/apps/kulala-fmt) is a command-line tool for formatting .http files,
ensuring consistent style and readability across your API definitions.

### VS Code

[Kulala.vscode](/apps/kulala.vscode) is in early development,
with a focus on a seamless experience for users familiar with the VS Code environment.

## Desktop apps

### Kulala Desktop

[Kulala Desktop](/apps/kulala-desktop) is currently in development.
It's electron-based,
with a focus on user-friendly features and UI,
including a visual request builder and response viewer.
